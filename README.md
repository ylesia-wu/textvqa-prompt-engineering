# TextVQA Prompt Engineering

This repository contains the code and outputs for our project on prompt engineering for TextVQA. We compare prompt strategies for two vision-language models: Qwen2.5-VL-3B-Instruct and LLaVA-Phi-3-mini.

## Project information

**Title:** Prompt Engineering for Text-Based Visual Question Answering  
**Course:** SHBT 261: AI in Medicine  
**Institution:** Harvard University  
**Term:** Spring 2026
